## Правила работы с Git

* git init - инициализация Git
* git add  - добавление файла в отслеживание
* git commit - закрепление файла для ослеживани
* git clone - копирование репозитария с удаленного термнала в локальный
* git push - передача локального репозитария в удаленный
* git pull - скачивание репозитария
* git download - загрузка репозитария
* git remote  - загрузка репозитария
* git pull request - запрос подтверждения загрузки локальных изменений.
